# Rustastic Password

`rpassword` makes it easy to read passwords in a console application on all platforms, Unix, Windows, WASM, etc.

`rpassword` is made available free of charge. You can support its development through [Liberapay](https://liberapay.com/conradkleinespel/) ðŸ’ª

## Usage

Add `rpassword` as a dependency in Cargo.toml:

```toml
[dependencies]
rpassword = "7.2"
```

See examples and docs at [https://docs.rs/rpassword](https://docs.rs/rpassword).
